## WallPaper

fyne version v2 >=2.2

go version >=1.7

Install one of the available C compilers for windows, the following are tested with Go and Fyne:

- MSYS2 with MingW-w64 - msys2.org
- TDM-GCC - tdm-gcc.tdragon.net
- Cygwin - cygwin.com
